Welcome to Lexophile

Quick setup — about a minute. You'll need Obsidian open with the Lexophile plugin enabled.

1Obsidian server URL

Where the Lexophile plugin is listening. The default works for most people.

2Choose an API token

A shared secret so only you can write to your vault. Don't leave this blank.

Then in Obsidian: Settings → Lexophile → API token — paste this exact value, then click Restart server.

3Verify the connection

Test that Chrome can reach Obsidian and the tokens match.

You're all set

Highlight a word on any webpage, right-click, and choose Add "<word>" to Lexophile.

You can change settings anytime from the Lexophile toolbar icon.
